Overview

What This PoC Does

Validates end-to-end on-device AI summarization triggered by simulated gaze hold, running entirely on the local Intel chip — no API, no cloud, no GPU required.

👁

Gaze Hold Trigger

Simulated with mouse hover (2 s threshold). When the user holds gaze on a paragraph, L1 visual enhancement + summarization activate simultaneously.

🤖

On-device AI

Phi-4 Mini (INT4 quantized, ~1.8 GB) runs via OpenVINO on Intel NPU → iGPU → CPU fallback. No internet required after first model download.

🔌

Native Messaging

Edge extension communicates with a local Python process via Chrome Native Messaging protocol (stdin/stdout JSON). Registered in Windows Registry.

⚡

Target: < 2s

On Intel Core Ultra 2 (Lunar Lake) NPU, summary latency is 0.9–1.5 s. Panther Lake expected to be faster. Acceptable threshold: < 3 s.

Gaze Tracking

Eye-Tracking Integration

Two interchangeable modes share the same trigger interface — switch between them without changing content.js or background.js.

🖱

Mouse Simulation (Default)

Hover over any paragraph for 2 seconds to trigger L1 + summarization. No webcam, no calibration. Works on any machine. This is the PoC demo mode.

👁

Webcam Eye Tracking (Real Gaze)

WebGazer.js maps webcam frames to screen coordinates using ML. After a 9-point calibration click, gaze accuracy is ±50–100 px — enough for paragraph detection.

# Manual host test (run in a separate terminal)
import subprocess, json, struct

proc = subprocess.Popen(
    ["python", "native_host.py"],
    stdin=subprocess.PIPE, stdout=subprocess.PIPE
)

def send(obj):
    data = json.dumps(obj).encode()
    proc.stdin.write(struct.pack("=I", len(data)) + data)
    proc.stdin.flush()

def recv():
    n = struct.unpack("=I", proc.stdout.read(4))[0]
    return json.loads(proc.stdout.read(n))

send({"action": "ping", "reqId": 1})
print(recv())   # → {"status": "ready", "reqId": 1}

send({"action": "summarize", "text": "AI Kwau is a browser extension...", "lang": "en", "reqId": 2})
print(recv())   # → {"status": "ok", "summary": "...", "reqId": 2}

import subprocess, json, struct, time

proc = subprocess.Popen(
    ['python', 'native_host.py'],
    stdin=subprocess.PIPE, stdout=subprocess.PIPE
)

def send(obj):
    data = json.dumps(obj).encode()
    proc.stdin.write(struct.pack('=I', len(data)) + data)
    proc.stdin.flush()

def recv():
    n = struct.unpack('=I', proc.stdout.read(4))[0]
    return json.loads(proc.stdout.read(n))

# V4: Ping
send({'action': 'ping', 'reqId': 1})
r = recv()
assert r['status'] == 'ready', f'FAIL V4: {r}'
print('V4 PASS:', r)

# V5: English summary
t = time.time()
send({'action': 'summarize',
      'text': 'Studies show over 1.1 billion people experience vision impairment globally. Small fonts and poor contrast remain the top barriers to digital accessibility for aging users.',
      'lang': 'en', 'reqId': 2})

assert r['status'] == 'ok' and len(r.get('summary','')) > 10, f'FAIL V5: {r}'
